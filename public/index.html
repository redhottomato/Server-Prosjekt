✅ Census API is running

This is a backend-only API developed as part of a Census course assignment.

The purpose of the application is to allow an Admin user to manage census participants and related data through a REST API.

How this application is used

There is no front-end for this application. All functionality is accessed and tested using Postman.

Authentication

Most endpoints are protected using HTTP Basic Authentication.

  • Username: admin
  • Password: P4ssword

Available endpoints

  • GET /status – JSON health check
  • GET /admin/test – Verify admin authentication
  • POST /participants – Create participant
  • GET /participants – List participants

Note for evaluators:
This API is hosted on Render and uses a MySQL database hosted on Aiven. All endpoints return JSON responses and are tested via Postman, as specified in the assignment description.
